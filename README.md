# 🛡️ PhishGuard v2

**PhishGuard v2** est une plateforme interactive et ludique conçue pour sensibiliser les utilisateurs aux risques du hameçonnage (phishing). À travers des scénarios réalistes, les utilisateurs apprennent à identifier les emails, SMS et sites frauduleux.

[![Live Demo](https://img.shields.io/badge/Live_Demo-Accéder_au_site-blue?style=for-the-badge&logo=render)](https://phishguard-2mpl.onrender.com/)

---

## 🚀 À propos du projet

Le phishing reste l'une des menaces de cybersécurité les plus courantes. PhishGuard v2 propose une méthode d'apprentissage active basée sur la **gamification** pour aiguiser les réflexes des utilisateurs face à ces tentatives d'attaques.

### ✨ Fonctionnalités principales

- **Scénarios immersifs :** Simulation réaliste d'interfaces (emails, navigateurs, SMS) pour tester la vigilance réelle.
- **Système de score :** Suivi de la progression et précision des réponses pour motiver l'apprentissage.
- **Mode sombre professionnel :** Interface moderne, type "SOC" (Security Operations Center) pour une expérience utilisateur confortable.
- **Navigation rapide :** Prise en charge des raccourcis clavier (`L` pour Légitime, `H` pour Hameçonnage) pour une fluidité maximale.
- **Feedback éducatif :** Explications détaillées après chaque erreur pour comprendre *pourquoi* un élément était frauduleux.
- **Responsive :** Design adaptable à tous les écrans.

## 📦 Installation locale

Pour faire tourner le projet en local :

**1. Clonez le dépôt :**

```bash
git clone https://github.com/Maxime288/PhishGuard-v2.git
```

**2. Accédez au dossier :**

```bash
cd PhishGuard-v2
```

**3. Lancez le projet :**

Ouvrez le fichier `index.html` dans votre navigateur, ou utilisez une extension comme [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) (VS Code) pour un rechargement automatique.
